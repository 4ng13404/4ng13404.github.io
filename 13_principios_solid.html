1.3 Principios SOLID

¿Qué son?

Son una especie de normas para programar mejor y evitar problemas en el futuro. Cada letra es un principio.

Recuerdo el primero: Single Responsability

Este consiste en que cada método solo puede tener una función, si hace más de una está incumpliendo el primer principio.

¿Cuál cumple con en principio SOLID 1?

Caso 2

Creo la misma clase, pero para ahorrarme líneas de código creo un solo método llamado "mecanismo" que haga todo lo anterior, así no me complico.
